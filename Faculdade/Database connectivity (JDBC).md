==JDBC== é uma API Java lançada em 1997 para acesso a bancos de dados relacionais através de SQL.
	- Inspirada no ODBC
	- A API vale para qualquer banco (dependente da disponibilidade de um driver).
	- Pacote padrão: ==java.sql==
	- Requer conhecer os comandos SQL
° Boa parte da implementação dos recursos presentes no paco java.sql é feita através das classes presentes nos drivers JDBC.
	- ==Analogia==: o pacote java.sql é um ==framework== que é instanciado pelos drivers associados a cada banco.
