Tema: Deseja-se fazer um sistema de controle acadêmico para a Universidade
GHFlusão, onde para cada aluno matriculado nas turmas das disciplinas deve haver um
cadastro.

período.
As turmas das disciplinas são ministradas pelos professores da universidade.
Apenas os alunos matriculados em turmas realizam provas das turmas das
disciplinas regularmente.
